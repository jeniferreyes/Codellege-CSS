Modelo de Cajas

El modelo de cajas se encarga de condicionar todas las páginas web. Cada caja es creada cuando se agrega un etiqueta en HTML.

En el modelo de cajas existen diferentes elementos que lo componen entre ellos están.

Contenido

El contenido, el texto, una imagen, el texto de los elementos de una lista.

Padding (Relleno)

El relleno en el modelo de cajas es el padding, este se refiere a el espacio que existe entre el borde y el contenido.

Border

El borde, (border), es como la linea que encierra el contenido

Magin (Margen)

Este elemento es el que va po fuera del borde, es la parte exterior del elemento.
